# Assignment 3 - Programs Documentation

## A3_task_1.py: Factorial Calculator

### Description
This program defines a function named `factorial` that calculates the factorial of a number using recursion.

### Functionality
- **Input**: User enters a number
- **Processing**: Calculates factorial using two methods:
- **Recursion**: Function calls itself with decremented value
- **Output**: Displays factorial results from both approaches

### Key Features
- Error handling for negative numbers
- Base cases for 0 and 1 (factorial = 1)
- Comparison between iterative and recursive approaches

### Example
```
Input: 5
Output:
factorial of 5 is: 120
```
